Bem vindo!

Esta é uma prévia do sistema de gestão escolar.

Primeiro, para usar qualquer recurso deste app deve-se definir um ou mais administradores gerais (masters). Esses administradores delegarão às contas para os diferentes setores da escola no sistema. Caso já tenha definido um administrador, apenas faça login. Para definir um administrador é necessário ter acesso direto ao console do banco de dados, por questões de segurança.
